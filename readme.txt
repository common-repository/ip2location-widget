Displays the geolocation information of the visitor who is visiting your website by ip address.

== Description ==

IP2Location Widget displays the geolocation information of visitors who is accessing your website, like IP address, country, region, city, ZIP code and much more. There are 3 designs available for selection.

This widget supports multilingual where you can personalize the geo-data of visitors from different countries:
English, 日本語, 简体中文, 繁體中文, Eesti keel, Malay, Dansk, Nederlands, Gaeilge, Português, Türkçe, Italiano, Tiếng Việt, Español, Svenska, русский язык, Deutsch, français, suomen kieli, česky, العربية and 한국어 are the languages supported in the widget.

This plugin frees you from the hassle to setup the database for IP geolocation information retrieval, as the information will be retrieved from the hosted server.

== Installation ==

Manual Installation (Use this method only if you are unable to install the plugin via the admin area)
1. Download the plugin zip file.
2. Unzip and upload the `ip2location-widget` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Configure the setting of plugin through the `Settings > IP2Location Widget` menu in WordPress.
